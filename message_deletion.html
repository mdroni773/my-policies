Message Deletion Feature

This feature allows the BIOM Management Bot to delete messages in the group based on admin instructions.

Features Included:

  • Admin can send a command to delete any unwanted message.
  • Sub-admins also have this power if granted permission.
  • Auto-delete welcome messages after 10 seconds.
  • Bot will wait 15 seconds before allowing next message (optional ON/OFF).
